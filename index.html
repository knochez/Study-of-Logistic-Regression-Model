Studying and Implementing Logistic Regression from Scratch

Introduction

This project aims to understand and implement Logistic Regression as a fundamental Machine Learning model. The objective was to explore how prediction, loss, and optimization interact during training, using theory and practice to build a complete model from scratch.

Methodology

The Logistic Regression model was implemented step-by-step using NumPy for mathematical operations and scikit-learn for dataset preprocessing. The process included:

  • Defining the hypothesis function: y-hat = σ(wᵀx + b)
  • Applying the Binary Cross-Entropy (BCE) loss to measure prediction error
  • Training the model via Gradient Descent to iteratively minimize the loss

All learning components—including the hypothesis, loss, and optimization steps—were implemented manually to fully understand how model training works internally.

Results

After training, the Logistic Regression model achieved an accuracy of approximately 97% on the Breast Cancer Wisconsin dataset.

The training process produced smooth loss and accuracy curves, showing consistent learning and convergence. The results demonstrated how optimization through gradient descent systematically improves performance compared to random parameter sampling.

Reflection

This project deepened the understanding of how parameters, loss, and optimization interact in Machine Learning. It provided practical insight into the mathematical and computational foundation of model training.

Building Logistic Regression from scratch helped connect theoretical principles to real implementation—laying the groundwork for more advanced topics like neural networks and deep learning.

Next steps: Future work may include studying regularization techniques or extending the model to handle multi-class classification problems.
